# habitTree

独学で4ヶ月Reactを勉強した成果物です。

## 概要

習慣の継続や目標の達成を活動記録として共有できるアプリです。

## Demo

[demoアプリ](https://habit-tree.web.app/)

## 機能一覧

- ユーザー登録機能
- Google 認証機能
- 習慣の作成機能
- 活動の振り返り機能
- 習慣の共有機能
- 習慣化のLevel機能
